# Detecting MitM ARP Poisoning Attacks using Gradient Boosting Machines

## 📌 Abstract
This research focuses on detecting Man-in-the-Middle (MitM) attacks caused by ARP poisoning in enterprise wireless networks. Traditional intrusion detection systems often fail to identify such attacks due to their dynamic and stealthy behavior.

This project proposes a machine learning-based approach using Gradient Boosting Machines (GBM) to improve detection accuracy and efficiency. The model analyzes network traffic features to classify normal and malicious activities, aiming to provide a scalable and real-time detection solution.
